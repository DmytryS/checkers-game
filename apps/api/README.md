# Checkers backend

## Install

    $ git clone git@github.com:DmytryS/checkersApp.git
    $ cd cd ~/Documents/checkersApp/apps/api
    $ npm install

### Configure app

Copy `config.json.sample` to `config.json` then edit it

## Start

    $ npm start

## Check coverage with istanbul

    $ npm run test:coverage-check

## Check lint

    $ npm run lint
